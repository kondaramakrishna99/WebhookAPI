Welcome to Webhooks Buddy

Useful commands

  • @webhook git authorize #for authorizing the app to use your account
  • @webhook git repo list #to list your repositories
  • @webhook git createhook reponame #create hook for specific repository
  • @webhook git deletehook reponame inthisthread #delete hook for specific repository
  • @webhook git deletehook reponame allthreads #delete hook for specific repository
  • @webhook git getallhooks forthischat #delete hook for specific repository
  • @webhook git getallhooks mine #delete hook for specific repository
  • @webhook help #to get above commands anytime.
